* **JSX Architecture:** Writing clean, semantic markup within JavaScript logic.

## üöÄ Getting Started

To run the main React application locally, follow these steps:

### Prerequisites
* **Node.js** (Version 18+ recommended)
* **npm** (Included with Node.js)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/FrankieWilson1/alx-fe-reactjs.git
   # (Note: The React application is nested within the alx-react-app folder)
   ```

2. Navigate to the project directory:
   ```bash
   cd alx-fe-reactjs/alx-react-app
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open your browser: Visit http://localhost:5173 to view the application.
